# Swastrik Properties

Swastrik Properties is a premium real estate web application designed to provide a seamless and sophisticated experience for property management. The platform serves as a high-end marketplace connecting buyers with exclusive real estate opportunities through a modern, responsive interface.

## 🌐 Web Platform Features

*   **Premium User Experience**: A sleek, high-performance interface with dark mode aesthetics and gold accents, optimized for both desktop and mobile browsing.
*   **Property Discovery**: Intuitive tools to browse and filter properties by type, price, and location, ensuring users find exactly what they are looking for.
*   **Dual-Portal System**: 
    *   **Agent Portal**: A comprehensive dashboard for real estate professionals to manage listings, upload high-quality property imagery, and track interactions.
    *   **Buyer Portal**: A personalized space for users to track their inquiries, manage their profiles, and save property interests.
*   **Interactive Engagement**: High-quality image carousels for property viewing and direct communication channels including WhatsApp and phone integration for immediate connectivity.
*   **Real-time Updates**: Dynamic data management ensuring that property listings and inquiry statuses are always up-to-date.

## 🎨 Design Philosophy

The platform follows a "Minimalist-Luxury" design system:
- **Color Palette**: Deep blacks and elegant gold accents to evoke a sense of premium quality and trust.
- **Typography**: Clean, modern sans-serif typography for maximum readability.
- **Visual Feel**: Use of glassmorphism and smooth transitions to create an immersive, high-end digital environment.
